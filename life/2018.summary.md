# 程序员 2018 年度总结
这一年里，你一共写了 34 万行代码

你热衷于使用组合键 ⌘ + C 和 ⌘ + V

你热爱学习，喜欢和同事一起分享你们两个都不懂的技术知识

你每天早上都会打开 Github 假装了解最新的技术知识

2018年，你说的最多的几句话是:
* 这个需求做不了
* 在我电脑上是好的啊
* 你重启下电脑试试

4.23 大概是很特别的一天
